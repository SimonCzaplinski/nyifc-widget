# NYIFC Match Schedule Widget

Auto-updating match schedule calendar for [NY International FC](https://www.nyintfc.com).

Displays schedules for all 3 NYIFC teams:
- **NYIFC II** — CSL Division 2
- **NYIFC III** — CSL Division 2 Reserve
- **NYIFC** — APSL Metropolitan Conference

## Usage

Add this to your website:
```html
<div id="nyifc-schedule-root"></div>
<script src="https://cdn.jsdelivr.net/gh/SimonCzaplinski/nyifc-widget@main/nyifc-widget.js"></script>
```

Schedule data is automatically updated twice daily at 6am and 8pm ET.
